# 律师助手

基于 Codex 桌面端的律师 AI 知识工作环境，开箱即用，知识越用越厚。

## 安装

1. 打开 Codex 桌面端，点击「+ New Project」
2. 选择一个空文件夹（如 `~/律师助手/`）
3. 新建 Thread，输入：

   ```
   帮我安装 https://github.com/你的用户名/lawyer-agent
   ```

4. 等待安装完成，关闭当前 Thread，新建一个 Thread 即可开始使用

## 使用示例

- 「民间借贷的利率上限是多少？」
- 「帮我审查这份合同：[粘贴合同内容]」
- 「分析以下案情：[描述案情]」
- 「查看我的知识库」

## 知识自成长

每次使用后自动提取对话中的法条、案由、裁判观点，存入个人知识库。
越用越懂你的业务。
