## 🎯 痛点 & 解决方案

### 律师日常工作中的真实痛点

| 痛点 | ❌ 现状 | ✅ 律师助手的方案 |
|------|--------|-----------------|
| **法条多记不住** | 靠记忆力或手动翻书，效率低、容易遗漏 | 自然语言提问，秒级定位对应法条原文，附带司法解释 |
| **类案检索成本高** | 上裁判文书网逐条翻阅，一个案子检索半小时 | 语义理解案情，自动匹配相似判例和裁判观点 |
| **私人知识沉淀难** | 做了十年案子，经验全在脑子里或分散的笔记本里 | 每次咨询自动归档——法条、判例、诉讼策略，存成结构化知识库 |
| **新领域上手慢** | 接到不熟悉的案由，需要从头查大量资料 | 公共知识库覆盖常见民事案由，开箱即用 |
| **审合同没模板** | 每份合同手动逐条写批注，缺少标准化流程 | 合同审查模板辅助，逐条分析风险点 |

- 🏢 **中小律所团队** — 统一知识体系，新人上手快
- 🏛️ **企业法务** — 合同审查、法律检索的日常工具
- 📚 **法学院学生** — 实习、论文研究的法律知识管理平台

## ✨ 核心能力

### 1️⃣ 智能法律检索

用自然语言提问，无需精确关键词：

```
你问：民间借贷利率上限现在是多少？
→ 引民法典第680条，结合最新LPR给出具体数字

你问：离婚时房子是婚前买的，女方能不能分？
→ 引民法典婚姻家庭编，区分全款/按揭、加名/未加名等场景

你问：债务人转移财产怎么办？
→ 引撤销权相关法条（民法典第538-542条），附带类案参考
```

### 2️⃣ 判决书深度分析（核心功能）

上传或粘贴法院判决书，自动完成 **10 维度分析**：

| 维度 | 分析内容 |
|------|----------|
| 案件基本信息 | 案号、案由、法院层级、审判程序、代理律师 |
| 争议焦点 | 核心争议点、事实摘要（300字） |
| 诉讼动机分析 | 导火索、原告诉求（表面+真实动机）、被告立场、风险因子 |
| 证据链分析 | 双方证据逐项分析及法院采信理由 |
| 法律路径 | 每条法律推理路径的事实→法条→结论 |
| 推理链条 | 事实A → 法条B → 结论C 的完整逻辑链 |
| 关键转折点 | 判决中的关键转折事件 |
| 法官裁判倾向 | 裁判风格、证据采信偏好 |
| 模式标签 | 事实/证据/策略模式标签 |
| 实战启示 | 当事人建议、律师策略、可复用论证、风险预警 |

> 分析完成后，结果自动存入个人知识库，下次同类案件直接调用。

### 3️⃣ 知识自成长引擎

这是本工具最核心的能力，也是与普通 AI 搜索的本质区别：

```
📥 第一次使用
你：民间借贷利率上限是多少？
→ 回复后自动存档：民法典第680条 + LPR最新数据
→ 个人知识库：+1 条知识点

📥 第二次使用  
你：借款合同怎么写？
→ 除了公共法条，还会参考「民间借贷」相关知识
→ 新增归档：合同形式、利率约定、保证条款
→ 个人知识库：+3 条知识点

📥 第 N 次使用
你：帮我审查这份借款合同
→ 综合历史积累知识 + 公共法条 + 用户画像
→ 分析更深入、答案更贴切
```

**知识库类型：**

| 类型 | 图标 | 说明 |
|------|------|------|
| 法条 | 📖 | 法律条文及司法解释 |
| 判例 | ⚖️ | 典型案例裁判观点 |
| 法律术语 | 📝 | 专业名词解释 |
| 模板 | 📋 | 合同审查、文书模板 |
| 案件分析 | 🔍 | AI 深度分析报告 |
| 个人笔记 | 📓 | 用户沉淀的个人经验 |

### 4️⃣ 用户画像与个性化引用

系统自动记录你关注的案由类型、常用法条、话题偏好。回答时会自然植入个性化引用：

```
「根据你之前也处理过的离婚后财产纠纷案例，本案中…」
「你之前引用过民法典第680条，本次涉及的利率问题需要结合最新LPR…」
```

> 不强插引用，只在强关联时自然引入。

### 5️⃣ 可视化知识图谱

随时说「查看知识库」，浏览器自动打开知识图谱页面：

- 📊 **知识统计** — 法条、判例、术语各多少条，趋势一目了然
- 🗺️ **关联网络** — 法条 ↔ 判例 ↔ 术语 的关联关系，交互式浏览
- 🔍 **搜索筛选** — 按关键词搜索，按类型筛选
- 📋 **节点详情** — 点击节点查看完整内容

## 🚀 安装指南

> **只需操作一次，后续开箱即用。**

### 环境要求

- ✅ Codex 桌面端（[下载地址](https://codex.ai)）
- ✅ Node.js >= 18（[下载地址](https://nodejs.org)）

### 安装步骤（约 1 分钟）

```
Step 1: 打开 Codex 桌面端，点击「+ New Project」
Step 2: 选择空文件夹（如 ~/律师助手/）
Step 3: 新建对话
Step 4: 输入指令：

   帮我安装 https://github.com/lebiai/lawyer-agent

Step 5: 等待「安装完成」提示
Step 6: 关闭当前对话，新建对话即可使用
```

安装过程自动完成：依赖安装 → 原生模块编译 → 嵌入模型下载 → 知识库初始化。

## 💡 使用场景示例

### 场景一：日常法律检索
```
你：民法典关于定金的规定是？定金和订金的区别？
→ 引民法典第586-588条，逐条解释区别

你：交通事故中，对方全责但没钱赔怎么办？
→ 引代位求偿权相关规定，分析实操路径
```

### 场景二：合同风险审查
```
你：帮我审查这份房屋租赁合同
→ 逐条分析：租金支付、押金返还、维修责任、违约条款
→ 标注高风险条款，给出修改建议
```

### 场景三：诉讼策略预判
```
你：客户借了50万，有借条、转账记录，对方说还了但没证据
→ 分析借贷关系成立要件、举证责任分配
→ 给出诉讼可行性评估和风险提示
```

### 场景四：类案研究
```
你：分析这个案子和之前处理过的民间借贷纠纷有什么异同
→ 检索个人知识库中的历史案件
→ 对比案由、争议焦点、判决结果
```
