# 📚 律师助手

基于 Codex 桌面端的民事诉讼 AI 助手。开箱即用，知识越用越厚。

---

## 它能做什么

帮你完成民事诉讼中的日常法律工作：

- **查法条** — 「民间借贷的利率上限是多少？」
- **审合同** — 「帮我审查这份买卖合同」
- **分析案情** — 「债务人有三套房产但拒绝还款，怎么处理？」
- **找类案** — 「民间借贷纠纷中法院一般支持多少违约金？」
- **记知识** — 每次使用后自动保存你的法律要点，越用越懂你

> ⚠️ 本工具仅处理**民事诉讼**相关事务。刑事、行政等非民事问题不会回答。

---

## 安装

> 只需要操作一次，后续开箱即用。

**准备工作：**
- 电脑上已安装 Codex 桌面端
- 电脑已安装 Node.js（如不确定，安装时会有提示）

**安装步骤：**

1. 打开 Codex 桌面端，点击「+ New Project」
2. 选择一个空文件夹（比如 `~/律师助手/`）
3. 新建对话，输入以下指令：

   ```
   帮我安装 https://github.com/lebiai/lawyer-agent
   ```

4. 等待屏幕出现「安装完成」的提示
5. 关闭当前对话，新建一个对话即可开始使用

全程大约 1 分钟。

## 使用示例

```
你问：民间借贷的利率上限是多少？
→ 助手会引用民法典第680条，结合最新LPR给出具体数字

你问：分析一下这个案情：张三借了李四50万，约定年利率24%，现在李四不还
→ 助手会分析借贷关系是否成立、利息是否合法、诉讼策略建议

你问：查看知识库
→ 展示你积累的所有法律知识，按类型分组
```

## 知识自成长

这是本工具最核心的能力。

每次你咨询完一个问题，助手会自动从对话中提取关键信息——引用过的法条、分析过的判例、总结出的诉讼策略——全部存入你的**个人知识库**。

举个例子：

```
第一次用：问「民间借贷利率」
    → 助手存入：民法典第680条、LPR相关解释

第二次用：问「借款合同怎么写」
    → 助手不仅搜公共法条，
      还会参考你上次积累的民间借贷知识

越往后，答案越精准，越贴合你的业务
```

每次回答末尾会显示本次归档了多少条知识点，你随时可以说「查看知识库」来管理。

## 可视化知识图谱

项目目录中的 `kb-viz.html` 是一个知识图谱可视化工具，双击在浏览器中打开即可使用：

- 📊 **知识统计** — 法条、判例、术语各多少条
- 🗺️ **知识图谱** — 法条 ↔ 判例 ↔ 术语 的关联关系，交互式浏览
- 🔍 **搜索筛选** — 按关键词搜索，按类型筛选
- 📋 **节点列表** — 点击跳转到图谱中的对应节点

使用方式：点击「选择数据库」，加载 `mcp/knowledge-server/data/knowledge.db` 或 `seed.db`。

## 可视化知识图谱

项目提供知识图谱可视化工具，展示法条、判例、术语之间的关联关系。

**首次使用：**
```bash
python3 scripts/generate-viz.py
```
然后双击 `kb-viz.html` 在浏览器中打开。

**后续更新：** 当知识库内容变化后，重新运行上述命令即可刷新图谱。
