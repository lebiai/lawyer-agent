# 法条引用规范

## 核心规则

法条引用必须写**标准法律全称+阿拉伯数字条号**。禁止任何描述性文字、推导性标注、相对引用。

## 正确示例

| 正确 | 说明 |
|------|------|
| 中华人民共和国民法典第一千零七十九条 | 法律全称+中文数字条号 |
| 中华人民共和国民法典婚姻家庭编的解释(一)第七十八条 | 司法解释全称+条号 |
| 中华人民共和国劳动合同法第四十七条 | 单行法全称+条号 |
| 中华人民共和国民事诉讼法第六十七条 | 程序法全称+条号 |
| 中华人民共和国公司法第一百八十二条 | 商事法全称+条号 |

## 绝对禁止的格式

| 错误 | 问题 |
|------|------|
| 民法典第1079条 | 未写"中华人民共和国"全称，用"第"替代"第X条"结构 |
| 同上 / 同解释 / 同前 / 如上 | 相对引用，依赖上下文 |
| 《民法典》1079条 | 用书名号替代全称 |
| 根据法律规定 | 模糊引用，没有具体法条 |
| 无直接法律依据 | 这是分析性结论，非法条引用 |

## 关于法条引用的约束

- 法条必须来自判决书原文，不得自行引入判决书中未出现的法条
- 如果判决书中的法条引用不规范（如只写了"民法典1079条"），输出时**仍按全称格式输出**："中华人民共和国民法典第一千零七十九条"
- 如果判决书未引用任何法条，`法律路径`章节标注"判决书未引用具体法条"
