# 证据相关规则

## 证据分类标准（7种）

| 分类 | 说明 | 示例 |
|------|------|------|
| 书证 | 以文字/符号/图案表达思想的文件 | 合同、借条、协议书、房产证、结婚证、营业执照、发票 |
| 物证 | 以其存在/外形/质量/特征证明事实的物品 | 房产标的物、侵权工具、瑕疵产品实物 |
| 电子数据 | 以数字化形式存储/处理/传输的信息 | 微信聊天记录、转账截图、电子邮件、短信记录、银行流水 |
| 证人证言 | 证人当庭或书面陈述 | 证人出庭作证笔录、书面证人证言 |
| 鉴定意见 | 专业机构出具的鉴定结论 | 笔迹鉴定、价格评估报告、审计报告、DNA鉴定 |
| 勘验笔录 | 法院/执法人员现场勘验记录 | 现场勘验图、勘验检查笔录 |
| 当事人陈述 | 当事人对案件事实的陈述 | 庭审笔录中的当事人陈述、质证意见 |

## 证据名清洗规则（7条）

证据名提取时遵守以下规则：

1. **去掉《》书名号** — `《离婚协议书》` → `离婚协议书`
2. **去掉前置修饰** — 去掉"原告/被告的""提交的""出具的""手写的""制作的"等前缀
3. **去掉位置信息** — 去掉具体房号（801房/2栋/3单元）、街道门牌号
4. **去掉案号** — 去掉文书中的案号如`(2025)粤01民终19940号`、`鲁0611民初`等
5. **去掉冗余后缀** — 去掉"复印件1份""原件""复印件"等
6. **去掉无意义单字词** — 不单独保留"无""有""陈述""证言""证明""证据"等单一词汇
7. **长度不少于2个汉字**

### 清洗示例

| 原始 | 标准化后 |
|------|---------|
| 801房房产证原件 | 房产证 |
| 彭某手写的《保证书》《悔过书》 | 保证书、悔过书 |
| 原告提交的微信转账记录截图1份 | 微信转账记录 |
| 福山区人民法院鲁0611民初民事判决书 | （应作为案由识别而非证据名） |
| 中国建设银行流水明细（2023年1月-12月） | 银行流水明细 |

## 法院采信评价标准

| 评价 | 含义 | 适用场景 |
|------|------|---------|
| fully_admitted | 法院完全采信 | 法院明确认定该证据"真实、合法、关联，予以确认" |
| partially_admitted | 部分采信 | 证据真实性认可但对证明目的不完全采纳 |
| rejected | 未采信 | 法院明确以"真实性无法确认""与本案无关""超过举证期限"等理由不予采信 |
