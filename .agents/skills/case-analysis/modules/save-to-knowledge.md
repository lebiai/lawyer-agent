# 模块：保存到知识库

## 归档规则

- **生效法律文书**（判决书/裁定书/调解书）→ 自动调用 `store_knowledge(type: case_analysis)`，title 为案由+案号，content 为完整分析报告
- **非生效法律文书**（仅案情描述/咨询）→ 不存储到知识库

## 写入内容

| 字段 | 内容 |
|------|------|
| title | 「[案由]案分析—[案号]」如「离婚纠纷案分析—（2026）甘3027民初24号」 |
| content | 完整分析报告（第1-10章全文），含用户问题和法律分析 |
| type | `case_analysis` |
| tags | 案由标签 + 模式标签 |
| reference | 案号 |

## 不存储的内容

- ❌ 判决书原文
- ❌ 用户个人信息
- ❌ 对话历史

## 保存后反馈
"📎 案件分析已存入知识库（输入「查看知识库」可管理）"
