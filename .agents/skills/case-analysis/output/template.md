# 输出模板

报告严格按以下 10 章结构输出。

```
═══════════════════════════════════════
  案件深度分析报告
═══════════════════════════════════════

📋 一、案件基本信息
  案号：[案号]
  案由：[案由]
  法院：[法院全称]
  审判程序：[一审/二审/再审]
  判决结果类型：[类型]
  代理律师：
    原告方 → [姓名]（[律所]）
    被告方 → [姓名]（[律所]）

⚖️ 二、争议焦点
  核心争议：[核心争议]
  事实摘要：[300字以内]

🔍 三、诉讼动机分析
  导火索事件：[原文标注]
  原告诉求（表面）：[原文]
  原告真实动机：[⚠️ 推断/原文]
  被告立场：[① ② ③]
  本案对律师的参考价值：[分析]
  风险因子：
  · [风险点]：[原文]

🗂️ 四、证据链分析
  原告证据：
  · [证据名]（[采信状态]）—— [举证目的]
    法院理由：[原文]
    证据权重：[高/中/低]
    证据分类：[分类]
    关键证明点：[一句话]
  被告证据：
  · [同上格式]
  举证责任分析：[分析]

📜 五、法律路径
  [路径A]
  · 触发事实：[原文]
  · 引用法条：[全称第X条]
  · 法条引入位置：[判决书位置]
  · 引入原因：[原因]
  · 适用推理：[推理]
  · 结论：[原文]
  · 依赖关系：[依赖/独立]

🔄 六、推理链条
  [箭头串联推理脉络]

🔄 七、关键转折点
  · [转折1]：[描述]（判决书位置）

👨‍⚖️ 八、法官裁判倾向
  裁判风格：[描述]
  证据采信偏好：[描述]
  同类案经验参考：[描述]

🏷️ 九、模式标签
  事实模式：[标签]、[标签]
  证据模式：[标签]、[标签]
  策略模式：[标签]、[标签]

💡 十、实战启示
  当事人可以怎么做：[基于本案]
  律师可以怎么用：[策略/取证/程序]
  可复用要点：[论证逻辑]
  风险预警：[趋势/注意]

═══════════════════════════════════════
```
