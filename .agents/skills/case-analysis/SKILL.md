# 民事案件分析 Skill

当用户提供案件材料要求分析时执行。这个 Skill 定义的是「如何做案件分析」的方法论，而不仅仅是输出格式。

---

## 核心原则

### 1. 分析必须完整
13 个维度逐项分析，**一个都不能少**。信息不足的维度标注「待补充」，不得跳过或合并。

### 2. 判断是否归档
- 用户提供了**法院判决书/裁定书/调解书**等已生效法律文书 → 存入知识库（`type: case_analysis`）
- 用户仅描述案情或咨询 → **不存储**，仅 `log_conversation(stored: false)`

### 3. 严格区分「分析」和「猜测」
- 能从材料中读到的信息 → 明确引用来源
- 需要推理的信息 → 说明推理依据
- 完全不知道的 → 标注「待补充」

---

## 13 维度的分析方法

### 一、当事人信息
- 从文件中提取原被告名称，注意核对是否与案件主体一致
- 如有多名当事人，全部列出并注明各自诉讼地位
- 缺失时标注「待补充」

### 二、案由确定
- 根据《民事案件案由规定》确定最精确的案由层级
- 方法：先确定法律关系大类（合同/侵权/物权/婚姻家庭/继承/劳动争议等），再确定具体案由
- **必须同时给出「法律关系」和「具体案由」**，两者不可偏废

### 三、基本事实
- 按时间线梳理关键事实节点，每个节点标注日期
- 区分「已查明事实」（材料中明确记载）和「主张事实」（一方陈述）
- 关键事实应是影响法律判断的核心要素，剔除无关细节

### 四、争议焦点
- 争议焦点应是**法律层面的争议**，而非事实层面的分歧
- 方法：当事人主张 → 对方抗辩 → 提取法律争议点
- 每个焦点都要明确指出双方各自的立场和依据
- 一般 2-4 个焦点，不宜过多

### 五、证据分析
- 对每份证据评估「三性」：真实性、合法性、关联性
- 区分「本证」（证明己方主张）和「反证」（反驳对方主张）
- 明确举证责任分配，指出哪一方对哪个事实承担举证责任
- 缺失证据要评估对案件的影响程度（致命/重要/一般/轻微）

### 六、法律适用
- 引用法条必须精确到条款项（如：民法典第680条第1款）
- 分析法条构成要件是否满足，逐一对照案情
- 如有司法解释，优先适用具体司法解释
- 注意法律修订情况，确保引用的法条当前有效

### 七、类案参考
- 优先使用 `search_knowledge` 搜索到的类案
- 如果没有完全匹配的类案，搜索最接近的案由类型
- 引用类案时说明案情相似点和不同点，以及参考价值

### 八、诉讼请求分析
- 逐项分析每个诉讼请求是否具有法律依据
- 涉及金额的计算必须明细化：本金 + 利息 + 违约金等
- 利息计算需注明：起算日、利率标准、计算方式
- 指出请求中可能被法院调整的部分（如过高违约金）

### 九、诉讼时效
- 民事诉讼一般时效 3 年（民法典第188条）
- 特殊时效：1年（如部分劳动争议）、4年（国际货物买卖合同）等
- 起算点：知道或应当知道权利受损之日起
- 中断事由：起诉、仲裁、请求履行、债务人同意履行等
- 判断标准：从起算点到当前是否超过时效期间，是否有中断事由

### 十、管辖分析
- 合同纠纷：被告住所地或合同履行地（有约定从约定）
- 侵权纠纷：侵权行为地或被告住所地
- 不动产纠纷：不动产所在地专属管辖
- **注意专门管辖**：如金融、海事、知识产权等

### 十一、诉讼策略
- 综合分析后给出推荐方案：诉讼/调解/仲裁/和解
- 诉讼请求金额需结合证据和法律规定给出建议范围
- 财产保全：评估是否有保全必要、保全范围和可行性
- 策略建议要有针对性，不能泛泛而谈

### 十二、风险评估
- 胜诉概率综合评估：法律依据充分性 + 证据完整性 + 类案倾向
- 执行风险：评估被执行人是否有可供执行的财产
- 成本估算：诉讼费（按标的额计算）+ 律师费 + 时间成本
- **风险提示要客观**，不能一味乐观也不能过度悲观

### 十三、结论建议
- 综合以上所有分析，给出明确的行动建议
- 建议要具体可行，不能空泛
- 明确优先级：紧急事项优先

## 格式要求

```
═══════════════════════════════════════════
📋 民事案件分析报告
═══════════════════════════════════════════

一、当事人信息（分析方法见上）
...

（13个维度依次排列，格式不限但内容必须完整）
═══════════════════════════════════════════
```

## 常见错误
- ❌ 遗漏维度（必须全部覆盖）
- ❌ 事实和法律混为一谈（先认定事实，再适用法律）
- ❌ 引用法条不精确到款/项
- ❌ 风险提示不客观或过于绝对
- ❌ 金额计算只给结果不给过程
- ❌ 「待补充」标注的内容明明可以从材料中推断出来
