# 维度3：诉讼动机分析

## 分析指引

### 导火索事件
什么具体事件引发了本案诉讼。**引用原文依据。**

### 原告诉求与真实动机
- **表面诉求**：起诉状列明的诉讼请求（引用原文）
- **真实动机**：表面诉求背后的真正利益诉求。⚠️ 标记为推断，除非判决书明确写了

### 被告立场
被告的核心抗辩观点。用① ② ③ 分段列出。被告缺席的注明"未到庭，未提交答辩状"。

### 本案对律师的参考价值
这个案子的分析对律师有什么实务启发（2-3句）。

### 风险因子
逐项列出诉讼中存在的风险点，每项附原文依据。

## 格式

```
导火索事件：[原文标注]

原告诉求（表面）：[原文]
原告真实动机：⚠️ [推断结论]（原文依据）

被告立场：
① [观点1]
② [观点2]

本案对律师的参考价值：[分析]

风险因子：
· [风险点1]：[原文标注]
· [风险点2]：[原文标注]
```
