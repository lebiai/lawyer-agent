# 维度9：模式标签

## 分析指引

输出标准化、可复用的短标签（3-6个字）。**每一类至少输出 2 个标签。**

### 分类

| 类型 | 说明 | 示例 |
|------|------|------|
| 事实模式 | 案件中的典型事实特征 | "被告缺席""分居满两年""婚前购房" |
| 证据模式 | 案件中出现的典型证据类型 | "微信聊天记录""银行流水""欠条原件" |
| 策略模式 | 诉讼中的典型策略行为 | "原告举证不足""被告自认""检察监督" |

### 约束
- 禁止写入案由名称（如"离婚纠纷"）
- 禁止写入完整句子或描述
- 同一概念在不同案件中必须用同一标签
- 标签不重复
