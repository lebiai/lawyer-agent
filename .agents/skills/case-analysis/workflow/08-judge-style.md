# 维度8：法官裁判倾向

## 分析指引

分析审判人员的裁判风格和偏好。**如实描述，不过度解读。**

### 分析维度

- **裁判风格**：程序严谨/注重效率/侧重书面/侧重庭审/
- **证据采信偏好**：重书面证据/重鉴定/对当事人陈述严谨/对证人证言宽松等
- **同类案经验参考**：基于本案反映出的裁判倾向（如"在夫妻债务认定上严格遵循司法解释的举证责任分配"）

### 约束
- 只能基于本案判决书内容推断，不得编造法官的"历史记录"
- 不确定的标注 ⚠️
- 判决书未体现的内容不写

### 示例

```
👨‍⚖️ 法官裁判倾向

法官：周毛草（审判员）

裁判风格：
- 程序严谨——依法适用普通程序、传票传唤要件齐全
- 侧重客观标准——以分居满两年等客观事实为依据裁定感情破裂
- 对缺席被告的处理简洁——不因被告缺席而延长等待

证据采信偏好：对官方书证（身份证、结婚证）认定标准宽松，依职权直接确认

同类案经验参考：本案是典型的"被告缺席+分居满两年"离婚判决，思路符合基层法院家事审判主流
```
