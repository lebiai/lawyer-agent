# 维度6：推理链条

## 分析指引

重建法院从事实认定到法律适用的完整思维过程。**不是法律路径的罗列，而是展示法院的思考脉络。**

### 写法

用箭头串联各环节：

```
事实A → 适用法条B → 得出中间结论C → 结合事实D → 适用法条E → 最终结论F
```

### 简单案件示例

```
确认婚姻关系（结婚证） → 查明分居满两年（被告离家） → 适用民法典第1079条
→ 认定感情破裂 → 准予离婚 → 处理费用承担
```

### 复杂案件示例（含再审）

```
一审/二审（错误）：
  借款合同无刘某某签字 → 不认定为夫妻共同债务 → 刘某某不承担还款责任

检察院抗诉后（新证据）：
  银行流水→ 300万到刘某某账户 → 购房合同 → 资金用于购房
  → 适用夫妻债务司法解释第3条 → 用于夫妻共同生活
  → 应认定为共同债务

北京高院再审：
  确认新证据效力 → 资金流向购房 → 刘某某应共同还款
  → 撤销一审二审 → 改判三人共同还款
```
