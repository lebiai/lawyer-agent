# 维度7：关键转折点

## 分析指引

识别判决书中**影响案件走向的关键事件或事实**。通常包括：
- 突现的关键证据（如检察院调取的新证据）
- 关键证人的证言
- 当事人自认
- 法院对特定事实的自由裁量
- 程序上的重要节点（如撤诉、缺席、鉴定）

### 格式

```
· [转折1名称]：[描述]
  原文标注：[判决书位置]

· [转折2名称]：[描述]
  原文标注：[判决书位置]
```

### 示例

```
· 转折1：检察院调取公安机关卷宗
  检察机关从公安机关调取购房合同和银行流水，证明300万资金流向购房，
  这是原审当事人无法自行获取的关键证据（判决书第1-2页）
```
