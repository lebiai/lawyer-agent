# 维度2：争议焦点

## 分析指引

### 核心争议
用一句话概括本案双方当事人围绕的核心法律争议。

**示例：**
- "婚前购房婚后共同还贷，离婚时房屋增值部分是否属于夫妻共同财产、应如何分割"
- "夫妻一方以个人名义借款用于购房，该债务是否构成夫妻共同债务"

### 事实摘要
法院认定的基本事实，300字以内。**只写原文明确认定的事实**，不加入分析推断。

写作结构：
```
当事人关系 → 关键时间节点 → 核心事件 → 法院查明结论
```

**示例：**
> 原被告于2018年登记结婚，婚后未生育。被告于2023年6月离家出走至今未归，分居已满两年。原告曾于2023年3月起离婚诉讼后撤诉（被告家人承诺劝返），但被告仍未回家。原告二次起诉离婚。
