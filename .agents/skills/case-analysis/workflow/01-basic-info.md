# 维度1：案件基本信息

## 必须提取的字段

- **案号** — 完整案号，含括号和法院代字
- **案由** — 精准到二级案由（如"离婚后财产纠纷"非"离婚纠纷"）
- **法院** — 全称（含地域层级，如"甘肃省夏河县人民法院"）
- **审判程序** — 一审/二审/再审（再审注明启动方式：当事人申请/检察院抗诉/法院依职权）
- **判决结果类型** — 全部支持 / 部分支持 / 全部驳回 / 维持原判 / 依法改判 / 调解结案 / 其他
- **代理律师** — 逐方列出：`[诉讼地位] → [姓名]（[律所全称]）`
  - 注意区分"委托诉讼代理人"和"辩护人"
  - 注意 `for_party`：明确谁代理谁（如"原告XXX的委托诉讼代理人XXX"中的对应关系）
- **合议庭** — 审判长、审判员、人民陪审员（如判决书载明）
- **书记员** — 姓名

## 约束
- 律师的律所名称要去掉角色头衔后缀（如"吉林良智律师事务所律师"→"吉林良智律师事务所"）
- 律师没有律所信息的注明"未载明"
- 当事人本人到庭、未委托律师的注明"无（本人到庭）"
