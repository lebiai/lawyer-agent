# 维度5：法律路径

## 分析指引

判决书中的每条法律推理路径单独分析，按以下结构：

```
[路径名称]
· 触发事实：[原文——什么事实触发了这条路径]
· 引用法条：[法律全称第X条]
· 法条引入位置：[判决书的哪个部分引入的（如"本院认为"段第X段）]
· 引入原因：[法院为什么在此处引入该法条]
· 适用推理：[法院如何将该法条适用于本案事实]
· 结论：[原文结论]
· 依赖关系：依赖路径A/独立
```

### 约束

1. **法条格式** — 必须写标准全称+条号，详见 `rules/law-citation.md`
2. **覆盖所有引用的法条** — 包括实体法和程序法（如缺席判决的程序依据）
3. **注意时间效力规则** — 如判决书适用民法典前的司法解释（如《关于审理涉及夫妻债务纠纷案件适用法律有关问题的解释》），应单独为时间效力规则列一条路径
4. **无引用法条** — 判决书未引用具体法条时，注明"判决书未引用具体法条"

### 示例

```
路径A：夫妻共同债务认定
· 触发事实：潘某将300万元转入刘某某银行卡，资金最终用于购房（判决书第5页）
· 引用法条：[最高人民法院关于审理涉及夫妻债务纠纷案件适用法律有关问题的解释]第三条
· 法条引入位置："本院认为"段第3段
· 引入原因：需判断借款是否构成夫妻共同债务
· 适用推理：潘某转款→用于购房（夫妻共同生活）→虽刘某某未签字→借款发生在婚姻存续期间→应认定为共同债务
· 结论：刘某某应承担共同还款责任
· 依赖关系：依赖检察机关新证据
```
