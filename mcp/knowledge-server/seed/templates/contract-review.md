# 合同审查要点清单

## 主体审查
- [ ] 签约方是否具备主体资格
- [ ] 是否完成必要的授权

## 标的条款
- [ ] 标的物/服务描述是否清晰
- [ ] 数量、质量、规格是否明确

## 价款条款
- [ ] 金额、币种、支付方式是否明确
- [ ] 发票类型及税率约定

## 履约条款
- [ ] 履行期限、地点、方式是否明确
- [ ] 交付/验收标准

## 违约责任
- [ ] 违约金比例是否合法（不超过实际损失的30%）
- [ ] 免责条款是否合理

## 争议解决
- [ ] 管辖法院/仲裁机构约定
- [ ] 适用法律

## 其他
- [ ] 保密条款
- [ ] 知识产权归属
- [ ] 合同变更/解除条件
- [ ] 通知送达条款
